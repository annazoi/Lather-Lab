# Terra & Lather | Artisanal Soap E-Shop Implementation

I have successfully implemented the design for **TERRA & LATHER**, a premium soap e-shop, using Next.js 14 (App Router) and Tailwind CSS. The implementation focuses on a minimalist, artisanal, and botanical aesthetic that feels high-end and earth-conscious.

## 🎨 Design System

### Color Palette
- **Off-white/Stone (`stone-50`)**: Used for primary backgrounds to create a clean, airy feel.
- **Charcoal (`stone-900`)**: Used for high-contrast sections and primary text.
- **Sage Green (`sage-600`)**: The primary brand accent, used for buttons, borders, and highlights.

### Typography
- **DM Serif Display**: Elegant serif headings for branding and section titles.
- **Inter**: Clean sans-serif for body text, navigation links, and UI labels.

## 🛠️ Tech Stack & Setup

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS with custom stone and sage color scales.
- **Icons**: Lucide React
- **Animations**: Framer Motion for smooth entrances and hover effects.

### How to Run Locally

```bash
# 1. Install dependencies
npm install

# 2. Run the development server
npm run dev
```

The site will be available at `http://localhost:3000`.
